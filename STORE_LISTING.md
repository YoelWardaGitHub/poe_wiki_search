## Name
PoeWiki Quick Search

## Summary (max 250 chars)
Search the Path of Exile Wiki (poewiki.net) straight from your address bar. Type "poe", hit space, and search — no clicks required.

## Category
Search Tools

## Description
Add a lightning-fast keyword search for the Path of Exile Wiki to your browser.

Just type **poe** in the address bar, press **Space**, then enter what you're
looking for — items, skills, mechanics, uniques, anything. Press **Enter** to jump
straight to the results on poewiki.net.

Features:
- Omnibox keyword: type "poe" + Space to search instantly
- Goes directly to poewiki.net's own search (exact page titles jump straight to the article)
- Open results in the current tab, or Alt+Enter for a new tab
- Zero configuration, no accounts, no tracking
- Lightweight and open source

This is an unofficial, fan-made extension and is not affiliated with or endorsed by
Grinding Gear Games or the Path of Exile Wiki.
